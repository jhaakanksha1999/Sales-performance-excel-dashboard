📊 Sales Performance Dashboard (Excel & Data Analysis)
A complete, analytics‑driven Sales Performance Dashboard built using Excel, PivotTables, and a structured 2025 sales dataset. This project demonstrates strong skills in data cleaning, KPI modeling, dashboard design, and business storytelling—ideal for showcasing practical data analytics capabilities to recruiters and hiring managers.

📁 Project Summary
This project analyzes sales performance across regions, salespeople, product categories, and months. The dashboard highlights revenue trends, profitability, target achievement, and top performers, enabling quick and informed decision‑making.

🛠️ How the Dashboard Was Built
1. Data Preparation
Cleaned column names and standardized data types

Validated date fields

Added calculated fields such as Profit Margin = Profit / Revenue

Ensured consistency across categories and product names

2. Pivot Table Modeling
Created pivot tables for:

Revenue by Region

Revenue by Salesperson

Revenue by Month

Revenue vs Target

Revenue by Category

These pivots serve as the analytical engine behind the dashboard.

3. KPI Calculations
Used Excel formulas to compute:

Total Revenue

Total Profit

Total Units Sold

Revenue vs Target %

Average Profit Margin

4. Dashboard Design
Built KPI cards using shapes and cell references

Inserted charts linked to pivot tables

Added slicers and connected them to all pivots

Applied a consistent color theme for readability and storytelling
